# 事件概述

触发——响应机制

```
<body>
    <button id="btn">唐伯虎</button>
    <script>
    //   点击一个按钮，弹出对话框
    //  事件组成的三要素： 事件源  事件类型  事件处理程序
    // (1) 事件源 ：事件被触发的对象，指谁，如 按钮
    var btn = document.getElementById('btn');
    // (2)事件类型 ：如何触发，什么时间，比如：鼠标点击(onclick，或者鼠标经过，或者键盘按下
    // (3) 事件处理程序：通过一个函数赋值的方式完成
    btn.onclick = function() {
        alert('点秋香');
    }
    </script>
</body>

```
