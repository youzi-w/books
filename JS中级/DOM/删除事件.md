# 删除事件（解绑事件）



```
<div>1</div>
<div>2</div>
<div>3</div>
```



1. 传统方式：

   ```
   var divs = document.querySelectorAll('div');
   divs[0].onclick = function() {
   	 alert(11);
   	// 1. 传统方式删除事件
   	divs[0].onclick = null;
   }
   ```

   

2. removeEventListener 删除事件  里面的 fn 不需要调用加小括号

   ```
   function fn() {
               alert(22);
               divs[1].removeEventListener('click', fn);
           }
   ```

   

3. detachEvent

   ```
   divs[2].attachEvent('onclick', fn1);
   function fn1() {
   	alert(33);
   	divs[2].detachEvent('onclick', fn1);
   }
   ```
